# hxy7 修改说明

## 最新更新（讯飞Spark API集成 + 登录页浅色主题）

### 讯飞Spark API集成
- **AI导游（visit阶段）**：主要讲述和介绍点位信息，生动有趣地讲述历史故事。
- **AI探究教练（research阶段）**：主要引导学生进行科学探究，通过问题启发思考。
- 优先使用讯飞Spark API（更适合中文），自动降级到OpenAI/DeepSeek（如果API调用失败）。
- 环境变量配置：`XFYUN_APPID`, `XFYUN_API_SECRET`, `XFYUN_API_KEY`
- 已配置讯飞Spark API Lite模型，支持1024 tokens输出。

### 登录页面改造
- **主题改为浅色**：从深色主题改为白色背景，更加清爽自然。
- **响应式布局**：
  - 手机（`sm`以下）：登录表单居中，宽度100%（max-width: 26.5rem）
  - 平板/桌面（`sm`以上）：登录表单靠右，保持在右侧1/3区域
- **背景优化**：使用淡蓝色渐变（不透明度大幅降低），保留背景图的视觉效果但不抢眼。
- **背景层组合**：
  1. 底层：蓝色渐变+背景图（降低不透明度）
  2. 中层：白色从右到左渐变叠层（增强可读性）
  3. 顶层：浅蓝色从上到下渐变

## 登录页
- 首页 `/` 已改为进入 `/login`，不再默认跳到学生端。
- 登录页改为背景层 + 前景登录卡片的双层结构，背景使用 `登录图片(1).png` 复制生成的 `legacy/next_v1/public/login-bg.png`。
- 前景登录模块改为半透明毛玻璃卡片，包含 logo、产品标题、账号输入框、密码输入框、登录按钮和体验账号提示。
- 登录文案统一为“账号 + 密码”，不再把页面交互表现成邮箱登录。

## 登录逻辑
- 演示模式登录接口继续使用 `/api/demo-login`，以 `account` 和 `password` 校验。
- 管理员账号显示和登录账号统一为 `admin`，默认密码 `123456`。
- 管理端创建账号时写入 `account` 字段，演示模式下可直接使用该账号和密码登录。
- Supabase 模式下保留底层邮箱映射，但前端和管理端均按“账号”呈现。

## 学生端
- 证据背包定位为草稿 / 过程性记录，不再显示教师批改状态。
- “加入学习档案”后才进入正式档案，学习档案逐条显示是否已批改、分数和评价。
- 上传图片证据改为读取真实文件 dataURL，证据背包、点位详情、学习档案、教师查看、优秀作品展示等位置都能显示真实图片预览。
- 点位完成 API 改为写入运行时数据，任务中心会拉取进度并刷新已完成状态。
- “矿坑结构观察”与“矿坑露天观景台”已对齐：推荐资源显示矿坑露天观景台，对应当前任务显示矿坑结构观察。

## 教师端
- 任务发布 API 取消只能发布到教师绑定班级的限制，支持多班级选择并真实写入 `class_ids`。
- 学生端任务列表按学生绑定班级过滤，非目标班级不会看到对应任务。
- 教师首页底部“案例工作台”“发布任务”按钮已删除，相关功能保留在顶部导航。
- “学生进度/作品与档案评价”改为“档案评价”，每个正式档案作品展示对应分数和评价。

## 管理端
- 删除页面上方独立“密码重置”区域。
- 每个账号行保留“重置密码”操作，点击后随机生成复杂密码，先弹窗显示新密码，再提交重置并弹出“重置成功”。
- 重置后的密码会写入演示运行时数据，可直接用于该账号登录。
- admin 账号显示为 `admin`，不再显示 `admin@demo.local`。
